使用 Gitbook 搭建 book page

2019-08-16

1 Gitbook是什么

GitBook是一个命令行工具（通过使用Node.js），用于使用GitHub / Git通过Markdown（或AsciiDoc）构建文档、页面以及书籍。本文将以构建一本书籍为例进行说明。

2 开始Gitbook

安装前请先安装Node.js和npm工具

2.1 通过npm安装

安装Gitbook最方便的的方法就是通过npm一条指令进行安装：

$ npm install gitbook-cli -g

gitbook-cli是一个在系统上安装和使用GitBook的npm包程序，自动安装所需版本的GitBook来构建书籍。
可通过gitbook-cli来切换gitbook的版本：

$ gitbook fetch beta

使用gitbook ls-remote可罗列出所有可用版本。

由于npm网络太慢，这里我采用了cnpm的淘宝镜像进行安装。
安装完成后测试指令：

$ gitbook -help

成功截图：

2.2 创建书籍

在创建之前新建我们想要保存这些构建文件的根目录，在该目录下打开命令行，使用初始化命令构建书本：

$ gitbook init

如果我们不想在本目录创建书籍，可以使用以下命令指定目录：

$ gitbook init ./directory

创建完成后会生成两个文件：

README.md 和 SUMMARY.md 是两个必须文件，README.md 是对书籍的简单介绍，SUMMARY.md 是书籍的目录结构。

2.3 预览、构建静态页面

创建完成后可通过以下指令预览：

$ gitbook serve


在浏览器打开http://localhost:4000/即可预览：

书籍制作完成后也可以直接构建成静态页面：

$ gitbook build ./repository --output=./outputFolder

构建完成后将会在目录生成相应的静态页面文件夹。

Gitbook 创建了书籍目录结构后，我们就可以向其中添加真正的内容了，文件的编写使用 markdown 语法，在内容修改过程中，每一次保存文件，gitbook serve 都会自动重新编译，所以可以持续通过浏览器来查看最新的书籍效果！

命令 build 和 serve 的参数为:

 -o, --output <directory>  #输出文件件, 默认为 ./_book
 -f, --format <name>       #产生的书籍的类型, 默认为静态站点, 可用的格式为: site, page, ebook, json
 --config <config file>    #配置文件, 默认为 book.js 或 book.json

3 配置

Gitbook允许使用灵活的配置自定义书籍，配置选项在book.json或book.js文件中填写，不熟悉JSON语法可以使用JSONlint等工具验证语法。

3.1 常规设置

  • root 包含所有书籍文件的根文件夹的路径，除外 book.json
  • structure 指定自述文件，摘要，词汇表等的路径。见3.2结构段落。
  • title 书籍的标题，默认值是从README中提取的。
  • description 您的书的描述，默认值是从README中提取的。
  • author 作者姓名。
  • isbn 这本书的ISBN。
  • language 本书语言的ISO语言代码，默认值为en。
  • direction 文字的方向。可以是rtl或者ltr，默认值取决于值language。
  • gitbook 应该使用的GitBook版本。使用SemVer规范并接受类似的条件">= 3.0.0"。

3.2 结构段落structure

  • structure.readme 自述文件名（默认为README.md）
  • structure.summary 摘要文件名（默认为SUMMARY.md）
  • structure.glossary 词汇表文件名（默认为GLOSSARY.md）
  • structure.languages 语言文件名（默认为LANGS.md）

3.3 插件Plugins

插件及其配置在book.json中指定。
从版本3.0.0开始，GitBook可以使用主题。

  • plugins 要加载的插件列表
  • pluginsConfig 插件配置

3.4 PDF选项

PDF输出可以使用以下选项中的一组选项进行自定义book.json：

  • pdf.pageNumbers 将页码添加到每个页面的底部（默认为true）
  • pdf.fontSize 基本字体大小（默认为12）
  • pdf.fontFamily 基本字体系列（默认为Arial）
  • pdf.paperSize 纸张尺寸，选项是'a0', 'a1', 'a2', 'a3', 'a4', 'a5', 'a6', 'b0', 'b1', 'b2', 'b3', 'b4', 'b5', 'b6', 'legal', 'letter'（默认是a4）
  • pdf.margin.top 上边距（默认为56）
  • pdf.margin.bottom 底部保证金（默认为56）
  • pdf.margin.right 右边距（默认为62）
  • pdf.margin.left 左边距（默认为62）
